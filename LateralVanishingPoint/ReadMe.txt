This folder contains the following files



----------------------PYTHON FILES---------------------------------------------------------
LateralVanishingPoint.py --> For solving the lateral vanishing point

Light.py                 --> "Light" Class file used by "LateralVanishingPoint.py" 
LateralVPNight.py        --> For solving the lateral vanishing point during night time (still needs to be developed)
cars3.xml                --> XML file for HAAR based vehicle detector
annotate.py              --> For annotating the taillights manually. The annotated images and the annotation values (as a CSV file) are stored "annotated" folder

yolo_pipeline.py         --> For identifying the bounding box of the vehicle using YOLO DNN modules.
./Weights                --> This folder contains the DNN weights for YOLO pipeline
lane.py                  --> Module used by YOLO pipeline
visualizations.py        --> Files stores different visaulization functions used by YOLO pipeline. 
utils.py                 --> Configuration file where the weights are loaded

---------------------MATLAB FILES---------------------------------------------------------
LateralVP.m              --> Uses the csv file generated by (LateralVanishingPoint.py) and plots these lines to identify the average slope  
FindIntersection.m       --> Finds intersection of two lines given slope and intercepts.
DeriveLines.m            --> Derives line points given slope and intercept value.
